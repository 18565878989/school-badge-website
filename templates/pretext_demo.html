📊 Pretext vs 传统方案

⚠️ 传统 DOM 方式

  • 每次渲染触发重排重绘
  • 复杂文本性能下降明显
  • 容易出现文字跳动
  • 帧率不稳定

✨ Pretext 引擎

  • Canvas 预计算文本宽度
  • 纯算术计算定位
  • 零 DOM 操作
  • 稳定 60-120 FPS
⚙️ 工作原理
// 1. 预处理文本（一次性测量） const preparedText = preprocess( 'Hello世界！🎓', '16px Inter', { lineHeight: 1.6 } ); // 2. 纯算术计算布局（无 DOM 操作） const { height, lineCount, lines } = layout( preparedText, 300 // 容器宽度 ); // 3. 直接渲染结果，帧率拉满！
